Navegadores web

Un navegador web(en inglés, web browser) es un software, aplicación o programa que permite el acceso a la Web, interpretando la información de distintos tipos de archivos y sistios web para que estos puedan ser vistos

La funcionalidad básica de un navegador web es permitir la visualización de documentos de texto, posiblemente con recursos multimedia incrustados

Ejemplo de navegadores web

Safari: Safari es un navegador web de código cerrado desarrollado por Apple Inc. Está disponible para macOS, iPadOS e iOS

Mozilla Firefox es un navegador web libre y de código abierto desarollado por Linux, Android, iOS, macOS y Microsoft Windows coordinado por la Coorporación Mozilla y la Fundación Mozilla

Google Chrome: Google Chrome es un navegador web de código cerrado desarrollado por Google, aunque derivado de proyectos de código abierto
